# MEERAS Platform

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/erkinasadov-crypto/meeras-platform.git
   cd meeras-platform
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the project:
   ```bash
   npm run build
   ```

## Run Instructions

To start the application locally, use:
```bash
npm start
```
